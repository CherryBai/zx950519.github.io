## A+B和C (15)
题目描述
给定区间[-2的31次方, 2的31次方]内的3个整数A、B和C，请判断A+B是否大于C。
```
import org.omg.PortableInterceptor.INACTIVE;

import java.util.*;
import java.io.*;
import java.math.*;
public class Main{

    public static void main(String[] args) throws Exception{
        BufferedReader bufferedReader = new BufferedReader(new InputStreamReader(System.in));
        int size = Integer.parseInt(bufferedReader.readLine());
        int index = 1;
        while(size>0) {
            String[] data = bufferedReader.readLine().split(" ");
            double a = Double.parseDouble(data[0]);
            double b = Double.parseDouble(data[1]);
            double c = Double.parseDouble(data[2]);
            if(a+b>c)
                System.out.println("Case #"+index+": "+"true");
            else
                System.out.println("Case #"+index+": "+"false");
            index++;
            size--;
        }
    }

}
```

## 数素数 (20)
令Pi表示第i个素数。现任给两个正整数M <= N <= 10000，请输出PM到PN的所有素数。
```
import org.omg.PortableInterceptor.INACTIVE;

import java.util.*;
import java.io.*;
import java.math.*;
public class Main{

    public static void main(String[] args) throws Exception{
        BufferedReader bufferedReader = new BufferedReader(new InputStreamReader(System.in));
        String[] data = bufferedReader.readLine().split(" ");
        int m = Integer.parseInt(data[0]);
        int n = Integer.parseInt(data[1]);
        boolean[] tmp = new boolean[104730];
        for(int i=2; i<=104729; i++) {
            int factor = 2;
            for(int j=i*factor; j<=104729; j=i*factor) {
                tmp[j] = true;
                factor++;
            }
        }
        Vector<Integer> vector = new Vector<>();
        int index = 0;
        for(int i=2; i<=104729; i++) {
            if(tmp[i]==false) {
                index++;
                if(index>=m && index<=n)
                    vector.add(i);
            }
        }
        index = 0;
        for(int i=0; i<vector.size(); i++) {
            index++;
            if(i==vector.size()-1) {
                System.out.print(vector.get(i));
                System.out.println();
                break;
            }
            if(index<10) {
                System.out.print(vector.get(i));
                System.out.print(" ");
            }
            else {
                System.out.print(vector.get(i));
                System.out.println();
                index = 0;
            }
        }
    }

}
```
