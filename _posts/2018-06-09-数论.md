## 最大公约数&最小公倍数

&emsp;&emsp;对于最大公约数和最小公倍数的求解方法，利用辗转相除法(欧几里德算法)即可。先求出最大公约数，再求出最小公倍数。  

```
    // 循环法实现辗转相除
    public static long gcd_1(long a, long b) {
        long k = 0;
        do {
            // 得到余数
            k = a % b;
            // 根据辗转相除法,把被除数赋给除数
            a = b;
            // 余数赋给被除数
            b = k;
        } while(k != 0);
        // 返回被除数
        return a;
    }
    // 循递归法实现辗转相除
    public static long gcd_2(long a, long b) {
        if(b == 0) {
            return a;
        }
        if(a < 0) {
            return gcd_2(-a, b);
        }
        if(b < 0) {
            return gcd_2(a, -b);
        }
        return gcd_2(b, a % b);
    }
```
