## A+B和C (15)
题目描述
给定区间[-2的31次方, 2的31次方]内的3个整数A、B和C，请判断A+B是否大于C。
```
import org.omg.PortableInterceptor.INACTIVE;

import java.util.*;
import java.io.*;
import java.math.*;
public class Main{

    public static void main(String[] args) throws Exception{
        BufferedReader bufferedReader = new BufferedReader(new InputStreamReader(System.in));
        int size = Integer.parseInt(bufferedReader.readLine());
        int index = 1;
        while(size>0) {
            String[] data = bufferedReader.readLine().split(" ");
            double a = Double.parseDouble(data[0]);
            double b = Double.parseDouble(data[1]);
            double c = Double.parseDouble(data[2]);
            if(a+b>c)
                System.out.println("Case #"+index+": "+"true");
            else
                System.out.println("Case #"+index+": "+"false");
            index++;
            size--;
        }
    }

}
```

## 数素数 (20)
令Pi表示第i个素数。现任给两个正整数M <= N <= 10000，请输出PM到PN的所有素数。
```
import org.omg.PortableInterceptor.INACTIVE;

import java.util.*;
import java.io.*;
import java.math.*;
public class Main{

    public static void main(String[] args) throws Exception{
        BufferedReader bufferedReader = new BufferedReader(new InputStreamReader(System.in));
        String[] data = bufferedReader.readLine().split(" ");
        int m = Integer.parseInt(data[0]);
        int n = Integer.parseInt(data[1]);
        boolean[] tmp = new boolean[104730];
        for(int i=2; i<=104729; i++) {
            int factor = 2;
            for(int j=i*factor; j<=104729; j=i*factor) {
                tmp[j] = true;
                factor++;
            }
        }
        Vector<Integer> vector = new Vector<>();
        int index = 0;
        for(int i=2; i<=104729; i++) {
            if(tmp[i]==false) {
                index++;
                if(index>=m && index<=n)
                    vector.add(i);
            }
        }
        index = 0;
        for(int i=0; i<vector.size(); i++) {
            index++;
            if(i==vector.size()-1) {
                System.out.print(vector.get(i));
                System.out.println();
                break;
            }
            if(index<10) {
                System.out.print(vector.get(i));
                System.out.print(" ");
            }
            else {
                System.out.print(vector.get(i));
                System.out.println();
                index = 0;
            }
        }
    }

}

## 部分A+B (15)  
正整数A的“DA（为1位整数）部分”定义为由A中所有DA组成的新整数PA。例如：给定A = 3862767，DA = 6，则A的“6部分”PA是66，因为A中有2个6。现给定A、DA、B、DB，请编写程序计算PA + PB。 
```
import org.omg.PortableInterceptor.INACTIVE;

import java.util.*;
import java.io.*;
import java.math.*;
public class Main{

    public static void main(String[] args) throws Exception{
        BufferedReader bufferedReader = new BufferedReader(new InputStreamReader(System.in));
        String[] data = bufferedReader.readLine().split(" ");
        long a = Long.parseLong(data[0]);
        long da = Long.parseLong(data[1]);
        long b = Long.parseLong(data[2]);
        long db = Long.parseLong(data[3]);
        long apk = getNewNumber(a, da) + getNewNumber(b, db);
        System.out.println(apk);
    }

    public static long getNewNumber(long data, long factor) {
        long ans = 0;
        while(data>0) {
            if(data%10==factor) {
                ans = ans*10 + factor;
            }
            data /= 10;
        }
        return ans;
    }

}
```

##  组个最小数 (20)
给定数字0-9各若干个。你可以以任意顺序排列这些数字，但必须全部使用。目标是使得最后得到的数尽可能小（注意0不能做首位）。
```
import org.omg.PortableInterceptor.INACTIVE;

import java.lang.reflect.Array;
import java.util.*;
import java.io.*;
import java.math.*;
public class Main{
    public static void main(String[] args) throws Exception{
        BufferedReader bufferedReader = new BufferedReader(new InputStreamReader(System.in));
        String[] data = bufferedReader.readLine().split(" ");
        int[] data_int = new int[data.length];

        for(int i=0; i<data.length; i++) {
            data_int[i] = Integer.parseInt(data[i]);
        }
        String sum = "";int index = 0;
        for(int i=1; i<data.length; i++) {
            if(data_int[i]>0) {
                index = i;
                break;
            }
        }
        sum = sum + String.valueOf(index);
        data_int[index]--;
        for(int i=0; i<data.length; i++) {
            while(data_int[i]>0) {
                sum = sum + String.valueOf(i);
                data_int[i]--;
            }
        }
        System.out.println(sum);
    }
}
```

## 完美数列(25)
给定一个正整数数列，和正整数p，设这个数列中的最大值是M，最小值是m，如果M <= m * p，则称这个数列是完美数列。
```
import org.omg.PortableInterceptor.INACTIVE;

import java.lang.reflect.Array;
import java.util.*;
import java.io.*;
import java.math.*;
public class Main{

    public static void main(String[] args) throws Exception{
        Scanner scanner = new Scanner(System.in);
        int n = scanner.nextInt();
        int p = scanner.nextInt();
        Vector<Integer> vector = new Vector<>();
        for(int i=0; i<n; i++) {
            vector.add(scanner.nextInt());
        }
        Collections.sort(vector);
        int sum = 0;
        for(int i=0; i<n ;i++) {
            for(int j=i+sum; j<n; j++) {
                if(vector.get(j) > vector.get(i)*p)
                    break;
                sum++;
            }
        }
        System.out.println(sum);
    }
}
```

## 月饼 (25)
月饼是中国人在中秋佳节时吃的一种传统食品，不同地区有许多不同风味的月饼。现给定所有种类月饼的库存量、总售价、以及市场的最大需求量，请你计算可以获得的最大收益是多少。
```
import org.omg.PortableInterceptor.INACTIVE;

import java.lang.reflect.Array;
import java.util.*;
import java.io.*;
import java.math.*;
public class Main{

    public static void main(String[] args) throws Exception{
        Scanner scanner = new Scanner(System.in);
        int size = scanner.nextInt();
        int allOfNeed = scanner.nextInt();
        Vector<Cake> vector = new Vector<>();
        for(int i=0; i<size; i++) {
            Cake cake = new Cake();
            cake.setAmount(scanner.nextInt());
            vector.add(cake);
        }
        for(int i=0; i<size; i++) {
            Cake cake = vector.get(i);
            cake.setPrice(scanner.nextInt());
            cake.setFactor((1.0)*cake.getPrice()/cake.getAmount());
        }
        Collections.sort(vector, new Comparator<Cake>() {
            @Override
            public int compare(Cake o1, Cake o2) {
                if(o2.getFactor()>o1.getFactor())
                    return 1;
                else if(o2.getFactor()-o1.getFactor()==0)
                    return 0;
                else
                    return -1;
            }
        });

        double sum = 0;
        for(int i=0; i<vector.size(); i++) {
//            System.out.println(vector.get(i).getAmount()+"-"+vector.get(i).getPrice()+"-"+vector.get(i).getFactor());
            if(vector.get(i).getAmount()<=allOfNeed) {
                sum += vector.get(i).getPrice();
                allOfNeed -= vector.get(i).getAmount();
            }
            else {
                sum += vector.get(i).getFactor()*allOfNeed;
                allOfNeed = 0;
            }
            if(allOfNeed==0)
                break;
        }
        System.out.println(new java.text.DecimalFormat("#.00").format(sum));
    }
}

class Cake{
    public int amount;
    public int price;
    public double factor;

    Cake() {
        amount = 0;
        price = 0;
        factor = 0;
    }

    public int getAmount() {
        return amount;
    }

    public void setAmount(int amount) {
        this.amount = amount;
    }

    public int getPrice() {
        return price;
    }

    public void setPrice(int price) {
        this.price = price;
    }

    public double getFactor() {
        return factor;
    }

    public void setFactor(double factor) {
        this.factor = factor;
    }
}
