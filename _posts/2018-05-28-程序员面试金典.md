## 原串翻转
请实现一个算法，在不使用额外数据结构和储存空间的情况下，翻转一个给定的字符串(可以使用单个过程变量)。
```
import java.util.*;

public class Reverse {
    public String reverseString(String iniString) {
        return new StringBuffer(iniString).reverse().toString();
    }
}
```
